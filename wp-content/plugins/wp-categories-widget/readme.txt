Categories widget! Display lists of categories in website sidebar.

== Description ==

The "WP Categories Widget" is a very simple plugin to display the list of categories of any type of custom taxonomies.

Note : please don't forget to leave your valuable feedback and suggestions.

  **Send your query to [AUTHOR](mailto:raghunath.0087@gmail.com)**


  Video Tutorial : 
  
  https://youtu.be/pbsnErpu_8U
  

= `Features` = 
 * An option to show categories post count 
 * An option to hide child categories
 * An option to hide widget title
 * An option to show categories of any taxonomies type
 * An option to exclude/include categories of the taxonomies


= Addon Features =


 * Display Category as Drop Down
 * An option to show Category posts
 * An option to change background color
 * An option to change text color
 * An option to choose border style and border color
 * An option to add custom css

== Frequently Asked Questions ==

#Can I hide widget title?

 * Yes, You can hide widget title if you do not show in sidebar.
 
#Can I show list of category from my custom taxonomy?

 * Yes,You can choose your custom taxonomy for show list of their all categories.

#Can I exclude specific category from list?

Yes, We have given an option to exclude selected catgories from list.

#Can I Show only selected category from list?

Yes, We have given an option to select multiple catgories to show in widget.

#Can I hide child categoires from list?

Yes, We have given an option to hide child catgories from list.

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The categories post will be available on widegt page (Apperance >> Widget)
    WordPress administration page.
